Introduction

Welcome to the help page for the Optimizer module. This guide will help you understand its functionality and how to use it effectively.

Optimizer permits you to find the best reaction conditions for your procedure.

Before starting, you need to have a saved working procedure with the actual configuration (use Wizard to do it).

The Optimizer module is designed to help you fine-tune your procedure by identifying the most effective reaction conditions. Here's a step-by-step guide to using it:

ðŸ§­ Step-by-Step Instructions

  1. Create a new Optimizer
    Click the New button to begin a new optimization session.
    A prompt will appear allowing you to select the procedure you want to optimize. Make sure the procedure is already saved and properly configured using the Wizard tool.
  2. Explore the Procedure
    Once the procedure is loaded, you'll see a comprehensive list of all operations included in it.
    Each operation may contain values that can be optimized. These values are visually indicated as interactive buttons.
  3. Select Variables for Optimization
    Click on any of the highlighted buttons to open the variable selection panel.
    Choose the variables you wish to modify during the optimization process. These could include temperature, time, concentration, or other relevant parameters.
  4. Define Optimization Ranges
    By default, the Optimizer sets the minimum and maximum range for each variable to Â±20% of its current value.
    You can adjust these ranges manually if needed to better reflect your experimental constraints or goals.
  5. Choose Optimization Type
    Select the type of optimization you want to perform: Bayesian Optimization or Design of Experiments (DoE).
    Fill in all the required fields specific to the chosen method to ensure accurate and efficient optimization.
  6. Save Optimization File
    Save the current configuration in a .configuration file. A check of all the inserted parameters will be performed. Eventually you will receive warnings if some parameters are not set correctly.
  7. Run the Optimization
    After selecting variables and confirming ranges, initiate the optimization process.
    The module will evaluate different combinations within the defined ranges to identify the most promising conditions.
